Voorbeeld 1

  • Toon instructies
  • Toon eindresultaat
  1. Kopieër deze code in je eigen html-file en voeg een css file toe.
  2. Geef de hele pagina een grijze achtergrond kleur.
  3. Geef het item Honda een rode tekstkleur door gebruik te maken van het attribuut "id".
  4. Geef het item Renault & Volvo een blauwe tekstkleur door gebruik te maken van het attribuut "class".
  5. Geef de titel en de vetgedrukte tekst een groene tekstkleur.
  6. Geef alle elementen binnen de ongesorteerde lijst een oranje tekstkleur.
  7. Geef alle directe paragrafen binnen een div een witte achtergrond kleur.
  8. Geef de paragraaf die na de div komt de achtergrond kleur cyaan.

  10. Geef het eerste item in de ordered list de kleur cyaan.

Voorbeeld 2

  • Toon instructies
  • Toon eindresultaat
  1. Kopieër deze code in je eigen html-file en voeg een css file toe.
  2. Voeg een Font Awesome CDN toe aan je html-file.
  3. Geef de hele pagina de achtergrond kleur #b7e9f7.
  4. Voeg de font-family Anton & Titillium Web toe aan je html-file.
  5. Geef alle tekst op de pagina het font "Titillium Web" en een tekstgrootte van 16px.
  6. Geef alle titels op de pagina het font "Anton"
  7. Geef alle h2's een tekstgrootte 1,5x zo groot als het root element en onderlijn deze titels.
  8. Geef alle h3's een tekstgrootte 1,2x zo groot als het root element.
  9. Geef alle div's die de uitleg van een sport bevatten een zwarte rand van 3px dik.
  10. Rond ook de hoeken van deze rand af in de rechterbovenhoek met 20px.
  11. Zorg ervoor dat de inhoud binnen de rand 15px van de zojuist toegevoegde rand afstaat.
  12. Verwijder de standaard marges en paddings die worden toegepast op de elementen.
  13. Zorg ervoor dat de hele inhoud van de pagina 50px van de rand van het scherm afstaat.
  14. Geef de tweede h3 binnen elke sectie een marge aan de bovenkant van 10px.
  15. Verwijder de bolletjes voor elk item in de lijst.
  16. Geef elk eerste item in de lijst de kleur #D6AF36.
  17. Geef elk tweede item in de lijst de kleur #A7A7AD.
  18. Geef elk derde item in de lijst de kleur #A77044.
  19. Positioneer het icoontje binnen elke sectie in de rechterbovenhoek.
  20. Zorg ervoor dat alle afbeeldingen responsief worden.
  21. Zorg ervoor dat de tekst in de link naar boven niet getoond wordt.
  22. Zorg ervoor dat de link naar boven altijd op het scherm getoond wordt als je scrollt en positioneer deze in de rechteronderhoek.
  23. Geef de link naar boven een zwarte kleur en een witte achtergrond kleur.
  24. Zorg ervoor dat de link naar boven geen onderlijning heeft.
  25. Geef de link naar boven een ronde, zwarte rand van 2px.
  26. Geef alle h2's vanaf een schermbreedte van 800px een tekstgrootte 2,5x zo groot als het root element en onderlijn deze titels.
  27. Geef alle h3's vanaf een schermbreedte van 800px een tekstgrootte 1,5x zo groot als het root element.

Oefening 1

  • Toon instructies
  • Toon eindresultaat
  1. Kopieër deze code in je eigen html-file en voeg een css file toe.
  2. Voeg een Font Awesome CDN toe aan je html-file.
  3. Verwijder de standaard marges en paddings die worden toegepast op de elementen.
  4. Voeg de font-family Oswald toe aan je html-file.
  5. Geef alle titels op de pagina het font "Oswald".
  6. Centreer de h1-titel in het midden van de pagina.
  7. Zorg dat alle afbeeldingen op de pagina altijd volledig getoond worden.
  8. Geef alle secties een onderbroken, zwarte lijn van 3px dik.
  9. Zorg dat de content binnen de secties 10px van de rand afstaan.
  10. Zorg dat de inhoud van de pagina 10px van de viewport afstaat.
  11. Zorg dat er onder de h2-titels en de paragrafen 10px ruimte zit.
  12. Geef het camera icoontje een marge van 10px aan de linker kant.
  13. Zorg ervoor dat er binnen een sectie 2 afbeeldingen langs elkaar worden getoond wanneer de schermbreedte groter is dan 700px.
  14. Zorg ervoor dat er binnen een sectie 4 afbeeldingen langs elkaar worden getoond wanneer de schermbreedte groter is dan 1200px.
